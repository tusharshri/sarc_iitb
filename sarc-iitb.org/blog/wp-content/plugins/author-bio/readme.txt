=== Author Bio ===
Contributors: midrangeman
Donate link: http://www.geekyramblings.org/plugins/donate/
Tags: author,bio,biographical information
Requires at least: 2.2.0
Tested up to: 3.1.0
Stable tag: Trunk

This plugin will give you the ability to add the text from an authors
biographical information to the end of a post.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate it in the Plugin options

(pretty simple, eh?)

The biographical information box can be styled using the author-bio class.
